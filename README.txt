The Nowy Widget plugin allows you to create, manage, edit, and customize new Nowy app social content posts gallery layout.


== Description ==

The Nowy widget integrates with your website to display any posts directly from the Nowy social app into a single, customizable, and embeddable-anywhere feed from a designated business account. The Nowy widget helps travel bloggers, professional travelers, travel agencies, property management/marketing businesses, and related service providers build their own influential social community and connect to the largest supply of user content and experiences.

### Brand New Concept for Custom layout
 
The Nowy Widget is auto-scalable based on the website's layout and fits perfectly.

== Installation ==

<strong>From Your WordPress Dashboard</strong>
 
1. Go to Plugins ><strong>Add New</strong>
2. Search for <strong>Nowy Widget for WordPress</strong>
3. Click on <strong>Install Now</strong> Button
4. Click on <strong>Activate Now</strong> After Installed in the Backend 
 


== Frequently asked questions ==

= Do I need coding skills for Nowy Widget = 
 
Not at all. No code is needed for using the Nowy widget. It is completely compatible with any WordPress website or PHP-based website. It is also compatible with Elemontor. Any user is only to copy and paste the shortcode to their website to display Nowy's content from the Nowy app
 
= Is there any Option for the content display? = 
 
Yes, of course. You can easily include or exclude any content on Nowy widget's settings main page.
 
= May I Face any Compatibility Issues with my Theme? = 
 
No. but, you can share your theme name with us if you're facing compatibility issues.

= Where can I ask for help? =
 
Please reach out via the official email at info@nowy.io for any questions.
